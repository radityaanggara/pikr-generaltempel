## Persiapan & Menjalankan

### Frontend
1. Buka folder `pikr-website` di VSCode.
2. Jalankan Live Server pada `index.html` (atau buka file langsung di browser).
3. Pastikan file aset (`images/logo.png`, `images/hero.jpg`, dsb) ditambahkan.

### Backend
1. Pastikan Node.js terinstall (v14+ recommended).
2. Buka terminal dan jalankan:
   ```bash
   cd pikr-website/backend
   npm install
   npm start
